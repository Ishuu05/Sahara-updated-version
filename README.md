Sahara is an AI-powered, government-managed mobile application designed to bridge the gap between civilians and authorities during crises. The platform focuses on high performance in low-connectivity environments and inclusivity across all demographics.

## App in AI Studio: https://ai.studio/apps/cea44ee0-5706-430a-9a74-380307458d8d

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
